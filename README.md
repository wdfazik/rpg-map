# Атлас Забытых Земель

Интерактивная RPG-карта фэнтези-мира для **GitHub Pages**.

Рельеф генерируется **шумом Перлина** (fBm + domain warping). Кликайте по клеткам, исследуйте земли, находите ресурсы и текстовые квесты, скачивайте карту как PNG.

## Возможности

- Случайная (или по семени) генерация континентов
- Типы местности: океан, побережье, равнины, лес, холмы, горы, снежные пики
- **Туман войны** — неизведанные земли скрыты
- Исследование клеток: ресурсы в сумку или сюжетные квесты
- Экспорт карты в **PNG**
- Плавные анимации, тёмный fantasy-UI

## Локальный запуск

Откройте `index.html` в браузере или поднимите статический сервер:

```bash
# Python
python -m http.server 8080

# Node
npx serve .
```

Затем откройте `http://localhost:8080`.

## Публикация на GitHub Pages

### Вариант A: репозиторий только с этим сайтом

1. Создайте репозиторий на GitHub (например `fantasy-rpg-map`).
2. Загрузите содержимое папки `fantasy-rpg-map` в **корень** репозитория:

   ```
   index.html
   styles.css
   app.js
   perlin.js
   data.js
   README.md
   ```

3. **Settings → Pages → Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: `main` (или `master`), folder: **/ (root)**
4. Через 1–2 минуты сайт будет на  
   `https://<username>.github.io/<repo>/`

### Вариант B: проект в подпапке

Если сайт лежит в подпапке репозитория, в Pages укажите эту папку (или используйте GitHub Action / ветку `gh-pages` с содержимым корня сайта).

### Быстрый push (если git уже настроен)

```bash
cd fantasy-rpg-map
git init
git add .
git commit -m "Initial: interactive Perlin RPG map for GitHub Pages"
git branch -M main
git remote add origin https://github.com/<username>/<repo>.git
git push -u origin main
```

Затем включите GitHub Pages, как выше.

## Управление

| Действие | Как |
|----------|-----|
| Создать мир | кнопка «Создать мир» или Enter в поле семени |
| Случайное семя | 🎲 |
| Выбрать клетку | клик по карте |
| Исследовать | кнопка «Исследовать», **E**, или двойной клик |
| Туман войны | кнопка внизу карты |
| Скачать PNG | «Скачать PNG» |

## Структура

```
fantasy-rpg-map/
├── index.html    # разметка
├── styles.css    # UI и анимации
├── perlin.js     # шум Перлина + fBm
├── data.js       # локации, ресурсы, квесты
├── app.js        # генерация, исследование, экспорт
└── README.md
```

Чистый статический сайт: **без сборки**, без npm, без backend.
